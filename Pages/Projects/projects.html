    Conway's Game of Obstacles

    This game was inspired by a simulation program called "Conway's Game of Life", which was popular for a having a grid that updated it's own state based off a previous state which led to some insightful rules. My addition to this idea was in making it into a game in which each player movement updated the state of the grid (with the same rules). The goal is to collect all the flags in the level while avoiding the obstacles that spawn in the way.
