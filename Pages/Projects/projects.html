Conway's Game of Obstacles

This game was inspired by a simulation program called "Conway's Game of Life", which was popular for a having a grid that updated it's own state based off a previous state based on some insightful rules.

My addition to this idea was in making it into a game in which each player movement updated the state of the grid (with the same rules). The goal is to collect all the flags in the level while avoiding the obstacles that spawn in the way.
Blackjack

This version of Blackjack can be played from the terminal. It emulates the game to the best of my ability, and while I don't play, I think I got everything.

I was excited about this project because it was my first program in Python, which I was learning on the spot. I used familiar methods of Object Oriented Programming in Java to guide my efforts as well as resources I found online.

Hill Cipher

This was a school project I thoroughly enjoyed in my cybersecurity class at the University of Central Florida. It's a program that takes in a bunch of plaintext and encrypts it using a hill cipher.
